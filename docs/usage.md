## Usage (examples)

Open a Terminal and navigate to your solution folder.

-   Build a solution:

```cmd
Nb.exe solution
```
- Clean a solution:

```cmd
Nb.exe clean
```

- Run tests on a solution:

```cmd
Nb.exe test
```
- Create a staging build:

```cmd
Nb.exe staging
```
- Display available options:
    
```cmd
Nb.exe -c targets
```
